2. git status
3. git add *tên file* ( hoặc git add . (. để thêm tất cả các file))

5. git log (xem lịch sử commit)
6. git show *mã tem* (xem chi tiết 1 lịch sử commit)
7. git diff (để xem sự thay đổi của file đã modify)

8. gitk (mở GUI)
